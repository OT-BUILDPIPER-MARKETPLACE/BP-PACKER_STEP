* Clone the code available at [BP-PACKER-STEP](https://github.com/OT-BUILDPIPER-MARKETPLACE/BP-PACKER-STEP)
* Build the docker image

```
git submodule init
git submodule update
docker build -t registry.buildpiper.in/sonar-scan:0.1 . 
```
